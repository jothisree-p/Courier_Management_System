Courier Management System

Project Overview

The Courier Management System is a Java web application developed using JSP, Servlets, JDBC, and Oracle DB.
It allows users to:
Add courier shipment records
View a specific courier record
View all courier records
This project is built as per CPC assessment specifications.

Technologies Used

Java (JDK 8+)
JSP & Servlets (Jakarta / Tomcat)
JDBC
Oracle Database
HTML

Functionalities:

1. Add Courier Record

Validates input
Generates Courier ID
Inserts into database

Redirects to:
success.html on success
error.html on failure

2. View Courier Record

Search by Sender Name + Ship Date
Displays record in displayCourier.jsp
Shows message if not found

3. View All Courier Records

Displays all records in displayAllCouriers.jsp
Shows message if no data exists

How to Run

Import project into Eclipse / IntelliJ
Configure Tomcat Server
Set up Oracle DB and run table + sequence scripts
Update DBUtil with your DB credentials
Run the project
Open: http://localhost:8080/Courier_Management_System/menu.html
